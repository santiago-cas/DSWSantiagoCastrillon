¿Por qué existen las etiquetas en html?

Principalmente html5 es un lenguaje que sirve para interpretar paginas web en los navegadores. al principio los navegadores necesitaban una manera de identificar cada item o texto que se ponian en sus páginas web, por esto se inventaron las etiquetas, sirven para identifiar el tipo de item que va hacer trabajado, además de que le da una organización interno algo que no ve el usuario pero que es importante para el diseñador, ya que al manejar una buena estructura se hace más fácil el diseño del mismo.

¿Como se construyen las etiquetas en html5?

La sintaxis de las etiquetas en html5 son muy simples, no hay que saberlas todas para hacer una página web, pero si se necesita hacer una página web más compleja entonces es necesario tener un gran conocimiento de para que sirve cada una y en que momento utilizarlas. normalmente existen tres atributos que se utilizan en casi todas las etiquetas. pero primero vamos a ver un ejemplo de como es la estructura de una etiqueta. 

                <tag attribute1="value1" attribute2="value2" /></tag>

tag es el nombre de la etiqueta, attribute es el atributo que nosotro queremos llamar, y el valor que este va a tener. un ejemplo de esto es el siguiente. 

                <a href="https://youtube.com"  />Ir a youtube</a>

Una etiqueta creada así nos mostrará algo como esto

Ir a youtube

Ahora veamos los atributos más comunes en html

                id="texto": Este atributo nos indica que es único dentro del código de la pagina, sirve para identificar precisamente esa etiqueta dentro del CSS o JavaScript
                class="texto": Aveces es necesario no solo hacer un cambio por CSS o por JavaScript a una etiqueta sino a varias que tengamos dentro del codigo de la página,
                entonces la claseserá un identificador múltiple, que nos ayudará a ubicar más de una etiqueta.
                style="texto": Esta etiqueta nos va a servir para realizar un estilo estático dentro del html, el cual será aplicado a esta etiqueta en particular.
