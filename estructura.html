Ejemplo de una página web sencilla

Significado de la estructura

  • DOCTYPE

    • <!DOCTYPE html> La etiqueta doctype declara el tipo de documento, por lo que esta está señalando que este es un documento HTML.

  • HTML

    • <html></html>Delimita el documento HTML.

  • META

    • <meta name="description" content="Descripción de la WEB">

  • LINK

    • <link href="http://dominio.com/hoja-de-estilos.css" rel="stylesheet" type="text/css"/>Esta etiqueta contiene un link a una hoja de estilos externa, la cual se utilizará en este documento.

Uso para META
  • META CHARSET

    • <meta charset="UTF-8">Establece el tipo de codificación del documento.

  • META TITLE

    • <meta name="title" content="Título de la WEB">Contiene el titulo que se mostrará en los buscadores.

  • META DESCRIPTION

    • <meta name="description" content="Descripción de la WEB"> Es el texto que se muestra bajo el título en los motores de búsqueda.

  • BODY

    • <body></body>La etiqueta body delimita el cuerpo del documento y contiene todo aquello que podremos ver en nuestro navegador. Imágenes, textos, enlaces, video, etc…

  • HEADER

    • <header></header>

  • NAV

    • <nav></nav>Esta etiqueta sirve para delimitar el menú de la página, donde colocaremos los enlaces internos para movernos entre nuestras diferentes secciones del sitio web.

  • H1, H2, H3, H4, H5, H6

    • <h1>Título de la WEB</h1>
    <h2>CONTENIDO PRINCIPAL</h2>
    <h3>Testimonios</h3>
    <h4>Avisos legales</h4>
    Estas etiquetas establecen los encabezados. Se organizan por niveles siendo H1 el más importante y H6 el menos importante.

  • SECTION

    • <section></section>La etiqueta section engloba una sección de texto, imágenes y otros elementos que guardan cierta relación entre ellos. Normalmente siempre le podremos poner un título o encabezado.

  • ARTICLE

    • <article></article>La etiqueta article se suele encontrar dentro de una etiqueta section y sirve para dividir y ordenar los contenidos en su interior.

  • DIV

    • <div></div>Otra forma de dividir contenido para posteriormente aplicarle clases y modificar su estilo.

  • P

    • <p></p>Entre las etiquetas P colocaremos un párrafo de texto.

  • A

    • <a href="http://dominio.com/seccion2.html">IR SECCIÓN 2</a>Representa un enlace o hipervínculo.

  • ASIDE

    • <aside></aside>Contiene información no vital o que no está estrechamente relacionada con el contenido principal de la página como podrían ser banners de anuncios, citas o enlaces externos.

HREF

Dirección URL hacia la que apunta el enlace.

Etiqueta IMG

<img src="http://dominio.com/imagen.jpg" alt="paisaje">Esta etiqueta coloca una imagen en el documento mediante un enlace.

Atributos
  • ALT

    • Representa el texto alternativo, muy importante para que los bucadores puedan obtener información de la imagen.

  • SRC

    • URL donde se encuentra la imagen.

FOOTER

<footer></footer>Aquí encontraríamos el código perteneciente al pie de página, donde se suelen colocar los enlaces a textos legales, el copyright, etc.

Estructura básica de las etiquetas

Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen otras propiedades, como son los atributos y el contenido.

HTML define un total de 91 etiquetas, de las cuales 10 se consideran obsoletas. Sin embargo, una etiqueta por sí sola a veces no contiene la suficiente información para estar completamente definida. Para ello contamos con los atributos: pares nombre-valor separados por "=" y escritos en la etiqueta inicial de un elemento después del nombre del elemento. El valor puede estar encerrado entre "comillas dobles" o 'simples'. Existen, también, algunos atributos que afectan al elemento por su presencia en la etiqueta de inicio, como puede ser el atributo ismap para el elemento <img>.

Esta sería la estructura general de una línea de código en lenguaje HTML:

<tag attribute1="value1" attribute2="value2">content</tag>

Atributos básicos de las etiquetas

Atributo Describción
id="texto" Establece un indicador único a cada elemento
class="texto" Establece la clase CSS que se aplica a los estilos del elemento
style="texto" Aplica de forma directa los estilos CSS de un elemento
title="texto" Establece el título del elemento (Mejora la accesibilidad)
